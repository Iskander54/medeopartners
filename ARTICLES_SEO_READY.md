## Instructions d'Intégration

Ces articles sont prêts à être intégrés dans votre système de blog. Le contenu est optimisé SEO, structuré avec H1-H2-H3, et contient des appels à l'action vers votre cabinet.

### Pour intégrer via l'interface admin :
1. Se connecter à l'interface d'administration
2. Créer les catégories "Fiscalité" et "Création d'entreprise" si elles n'existent pas
3. Créer les articles avec le contenu fourni dans `scripts/init_db_and_create_articles.py`
4. Ajouter des images d'illustration pertinentes
5. Publier les articles

### Pour intégrer via script :
Le script `scripts/init_db_and_create_articles.py` contient tout le code nécessaire. Il faut d'abord résoudre le problème de connexion à la base de données.

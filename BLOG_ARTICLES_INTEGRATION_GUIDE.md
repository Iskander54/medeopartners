## 🔧 Méthodes d'Intégration

### Méthode 1 : Via Script Python (Recommandé)

Les scripts suivants sont disponibles :
- `scripts/init_db_and_create_articles.py` : Crée les tables et insère les articles
- `scripts/insert_articles_sql.py` : Insère directement via SQLite

**Problème actuel** : Les tables du blog n'existent pas encore dans la base de données.

**Solution** : Lancer l'application Flask une première fois pour créer les tables, puis exécuter le script d'insertion.

### Méthode 2 : Via Interface d'Administration

Si une interface admin existe, vous pouvez créer les articles manuellement en copiant le contenu depuis les scripts.

### Méthode 3 : Via SQL Direct

Si vous avez accès à la base de données SQLite, vous pouvez exécuter les requêtes SQL directement.

## 📊 Stratégie Blog Complète

### Calendrier Éditorial Recommandé

**Fréquence** : 2-3 articles par mois minimum

**Thématiques prioritaires** :
1. **Fiscalité** (Priorité 1)
   - ✅ TVA et obligations déclaratives (CRÉÉ)
   - Impôt sur les Sociétés (IS)
   - CVAE et CFE
   - Optimisation fiscale légale

2. **Création d'Entreprise** (Priorité 1)
   - ✅ Erreurs comptables première année (CRÉÉ)
   - Choix du statut juridique
   - Régimes fiscaux comparés
   - Obligations sociales

3. **Comptabilité Générale**
   - Plan comptable général
   - Écritures comptables
   - Amortissements
   - Clôture comptable

4. **Social et Paie**
   - Déclarations sociales
   - Gestion de la paie
   - Charges sociales

### Optimisations Recommandées

1. **Call-to-Action dans chaque article** :
   - "Besoin d'un accompagnement personnalisé ?"
   - Formulaire de contact intégré
   - Lien vers audit gratuit

2. **Lead Magnets à créer** :
   - Guide PDF : "Checklist des obligations fiscales 2024"
   - Calculateur : "Simulateur de charges sociales"
   - Template : "Modèle de facture conforme"

3. **Images d'illustration** :
   - Ajouter une image pour chaque article
   - Format WebP avec fallback JPEG
   - Alt tags optimisés SEO

4. **Partage Social** :
   - Boutons de partage LinkedIn, Facebook
   - Optimisation pour le partage (Open Graph)

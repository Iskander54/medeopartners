## 🔍 Instructions pour Google Search Console

1. **Connectez-vous à Google Search Console** : https://search.google.com/search-console

2. **Sélectionnez la propriété** : `https://www.medeo-partners.com`

3. **Utilisez l'outil "Inspection d'URL"** pour chaque URL ci-dessus :
   - Collez l'URL
   - Cliquez sur "Demander l'indexation"

4. **Ou utilisez le sitemap.xml** :
   - Allez dans "Sitemaps"
   - Soumettez : `https://www.medeo-partners.com/fr/sitemap.xml`
   - Les nouvelles URLs seront automatiquement détectées
   - Le sitemap inclut maintenant tous les articles du blog
